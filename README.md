# photon-web-recon
Photon is an **advanced automated reconnaissance tool** designed for comprehensive web intelligence gathering. 
This framework systematically extracts valuable OSINT (Open Source Intelligence) data from target websites, enabling security professionals to map digital footprints and identify potential attack vectors during penetration testing engagements.

Intelligence Collection Modules
- ** URL Harvesting**: Extract internal/external links recursively
- ** Contact Intelligence**: Email addresses & phone number discovery
- ** Document Mining**: PDF, Word, Excel, and other file enumeration
- ** Social Profiling**: Associated social media account detection
- ** Tech Stack Analysis**: Technology fingerprinting (CMS, frameworks, servers)
- ** Metadata Extraction**: Keywords, descriptions, and hidden metadata

Advanced Features
- **Recursive Crawling**: Multi-depth website traversal
- **Data Correlation**: Intelligent relationship mapping
- **Export Formats**: JSON, CSV, TXT outputs
- **API Integration**: RESTful API for automated workflows
- **Plugin Architecture**: Extensible module system
